# AI Agent Engineer Journey

Learning archive + experimentation playground + small portfolio.

This repository documents my end-to-end journey toward becoming an AI Agent Engineer. The goal is to keep everything:
- searchable (clear folder names + consistent docs)
- reproducible (logs, assumptions, and checklists)
- presentable (stage projects as milestones)

## Repository Scope

1. Learning route & progress tracking
2. Knowledge docs, concept explanations, and retrospectives
3. Single-point exercises (TypeScript, structured output, tool calling, workflow, etc.)
4. Stage projects (task assistant, research agent)

## How to Use This Repo

- `docs/` : topic notes and stage-oriented roadmaps
- `journal/` : daily learning logs (Day 01 starts the baseline)
- `exercises/` : focused practice tasks (small, testable outputs)
- `projects/` : stage projects (bigger deliverables with tracing/guardrails)
- `templates/` : reusable documentation templates for notes and projects
- `assets/` : diagrams and screenshots for explanations
