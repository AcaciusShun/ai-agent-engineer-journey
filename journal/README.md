# Journal (Daily Logs)

This folder stores daily notes as a chronological learning record.

## File Naming

- Use `day-XX.md` (two digits).
- `day-01.md` is the baseline day for the initial setup.

Example:
- `day-01.md`
- `day-02.md`

## Required Content per Day

Each `day-XX.md` should include:
- Current baseline (what you already understand)
- Objective for the day (what you attempted)
- Today’s outcomes (what you finished and what you learned)
- Problems encountered (errors, conceptual confusions, unexpected behavior)
- Tomorrow plan (next actions with a small scope)

## Cross-Linking Rules

- When you learn a concept, link it to the relevant `docs/` file.
- When you complete a practice task, link it to the relevant `exercises/` folder.
- When a daily log becomes a long decision record, consider moving it into `docs/99-retrospectives/`.
